# Changelog

## 0.2.0

- Added mean, +1 SD, and -1 SD reference lines calculated from each moving-average graph line.
- Added a per-analyte control to show or hide all mean/SD reference lines while preserving individual legend toggles.
- Added separate per-instrument charts with independent mean/SD reference-line controls.
- Added moving-average statistics tables for overview and instrument charts, including median as a non-graphed statistic.
- Moved instrument charts above analyte data and expanded the section by default.
- Added collapsed per-instrument additional graph sections with a normal distribution chart for moving-average values.
- Added mean and central 95% fitted-normal cutoff lines plus detailed distribution statistics tables.
- Added optional arbitrary fitted-normal confidence intervals with separate cutoff lines and complete statistics.
- Displayed fixed and custom interval statistics in side-by-side table columns.
- Added per-instrument raw-result time-series and raw-result normal distribution charts.
- Added mean +/-2 SD reference lines to all time-series charts.
- Added mean +/-2 SD values to moving-average and raw-result time-series statistics tables.

## 0.1.0

- Initial laboratory moving averages app.
- Supported CSV upload for analytes, instruments, timestamps, and results.
- Added separate and combined instrument visualizations.
- Added configurable moving-average point count.
- Added per-analyte axis controls.
- Added sample laboratory CSV data.
